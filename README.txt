** README **

The main interface of our project is a R Shiny app. In order to launch the interface, follow the following steps:

1. Open the Script.R file.

2. Before launching the app, make sure you have installed them by using following commands:

install.packages("sf")
install.packages("dplyr")
install.packages("shiny")
install.packages("DT")
install.packages("ggplot2")
install.packages("ggiraph")
install.packages("plotly")
install.packages("shinydashboard")
install.packages("shinyjs")

3. Run the R shiny app, and a dashboard will be prompted in your web browser.
